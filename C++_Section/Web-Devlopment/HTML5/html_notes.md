## HTML5 Basics

<hr>

- All html documents start with `<!DOCTYPE html>`.
- Used to tell browsers that this is an HTML document.
- Then HTML document itself begins with `<html>` and end with `</html>`

<hr>

```html
<!DOCTYPE html>
<html>
<body>
    <h1></h1>
    <p></p>
</body>
</html>
```
