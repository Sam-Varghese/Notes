# Declaring Variables

<hr>

- In order to declare variables, we firstly put `$` followed by variable name then `: ` and then its value
- Ex: `$main-fonts: Arial, sans-serif;`
- Use these variables like this: 

<hr>

```css
h1 {
  font-family: $main-fonts;
}
```
