# Git Branch

<hr>

## Creating New Branch

<hr>

```none
git branch <branch name>
```

<hr>

## Pushing Branch To Repo

<hr>

```none
git push -u origin <branch name>
```

<hr>

## Viewing Branches

<hr>

```none
git branch
```

OR

```none
git branch --list
```

<hr>

## Deleting A Branch

<hr>

```none
git branch -d <branch name>
```
